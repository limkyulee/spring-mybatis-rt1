# Admin Core API

Spring Boot REST API for Common Admin Management (User, Code, Menu, Role)

### ⚙️ Setting

1`.env` 파일 루트 경로에 생성 후 DB 정보 입력

### Tech
- Java 21
- Spring Boot 3.1.x
- MyBatis
- MySQL
- Gradle
- Lombok
