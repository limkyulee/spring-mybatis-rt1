# Admin Core API

Spring Boot + MyBatis 기반의 공통 관리 REST API 프로젝트입니다.  
(공통 코드 / 사용자 / 메뉴 / 권한 관리 기능 포함)

### Purpose
- 실무에서 자주 쓰는 관리 기능 구조 학습
- MyBatis 동적 SQL 사용 경험
- RESTful API 설계 및 MySQL 연동 연습

## Tech
- Java 17
- Spring Boot 3.1.x
- MyBatis
- MySQL
- Gradle
- Lombok
